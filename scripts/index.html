    • Sobre este material

    Este material didático foi criado inicialmente para ensinar/aprender a programação usando Processing modo Python, uma ferramenta livre de programação com a sintaxe de Python com os recursos de desenho e manipulação de imagens do Processing, dentro do Processing IDE, um projeto que não está mais ativo. O material desenvolvido entre 2019 e 2022 para esta ferramenta antiga, ainda está disponível aqui. Atualmente há várias outras possibilidades de combinar Python e Processing, e outras ferramentas que combinam Python e desenho ou modelos tridimensionas, veja abaixo com obter algumas delas:

    • A biblioteca py5, é o que recomendamos para a versão mais recente deste material, siga as instruções de instalação usando Thonny IDE + plugin.
    • O projeto pyp5js, combina pyodide e p5js, experimente uma variante do editor online que que tenta usar os nomes das funções da biblioteca py5.
    • Você pode explorar outras opções no quadro Processing + Python tools table
    • Ferramentas de desenho ou modelagem tridimensional com Python
